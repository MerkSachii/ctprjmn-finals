Project Charter: Green-blooded
Project Name Green-Blooded for the Generations to Come
Project Manager Mark Christian Sanchez
Goal The goal of this project is to entice and prepare my future offsprings for a Lasallian education. Measures should be done to persuade my offsprings to pursue their education in a Lasallian institution. A financial measure should be also made to cater to the expenses brought upon by studying.
Scope
    In-scope:
    1.) Lasallian schools
    2.) Financial planning
    3.) Persuasion techniques
    Out-scope:
    1.) Coercion
    2.) Non-Lasallian Schools (Ateneo etc.)
Risks
    1.) Offspring doesn't want a Lasallian education
    2.) Insufficient financial capacity
    3.) No more Lasallian institutions
Deliverables
    1.) Persuasion Plan
    2.) Financial Plan
    3.) Project Charter
    4.) Project Management Plan
    5.) Project Closure Report
    6.) Risk Assessment Plan
Budget P20,000,000
